基于多线程的网络聊天程序

根据这次的coding,对面向对象的其中两个特性有了直接的认识：
多态性：主要体现于一个类中的函数重载
封装性：利用面向对象编程的思想，主要是把以前的全局变量和全局函数，封装在类中声明为静态函数和变量。
